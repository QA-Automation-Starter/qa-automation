# Adding REST to QA Automation Project

Add `qa-jgiven-rest` dependency to your `pom.xml`

```xml
    <dependency>
      <groupId>dev.aherscu.qa</groupId>
      <artifactId>qa-jgiven-rest</artifactId>
      <version>...</version>
      <scope>test</scope>
    </dependency>
```

see tutorial #6 for a working example.
